# 按钮 Button

## 基本用法

按钮分为 主要按钮、次要按钮、虚线按钮、线形按钮和文本按钮五种。

<LiveEditor sourceCodePath="../../../../example/button/index.jsx"></LiveEditor>

Button 可以嵌入图标，在只设置图标而没有 children 时，按钮的高宽相等。

## 按钮状态

按钮状态分为 警告，危险，成功 三种，可以与按钮类型同时生效，优先级高于按钮类型。
